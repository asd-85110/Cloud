# Github 云盘
![melonlogo](melon.png)  
## 1.为什么我要使用Github作为云盘
A:其他的网盘太飞舞了,列举几个:  

1. bd网盘  
    必须登录,必须客户端,不开会员就限速等
2. 迅雷  
    和上面的差不多
3. 蓝奏云  
    原本以为很好,结果要开会员才有免密码的文件/目录,不利于种子文件
4. 123云盘  
    已经很好了,但是限制下载文件的大小  

所以,Github并不是最好的,但是现在只有它合适.
>注意:  
>这里没有完全列出云盘,  
>这只是一些样例而已
## 2.使用之前的提醒
- 安装`IDM/NDM/Motrix`等作为多线程下载工具,推荐`NDM`.`Motrix`,还有浏览器扩展
- 通过如Watt Toolkit|原本是Steam++|的工具加速访问
    >也可以不用  
    >没说科学上网|V*N|不行
- 访问下载界面
